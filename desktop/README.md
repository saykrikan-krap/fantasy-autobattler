# Desktop App

This is the Tauri + React desktop app skeleton.

## Prerequisites
- Node.js 20+
- Rust (stable toolchain)
- Tauri system prerequisites for your OS

## Run (development)
```
cd desktop
npm install
npm run tauri:dev
```

## Build (release)
```
cd desktop
npm install
npm run tauri:build
```

If you only want the web UI (no Tauri shell), you can run:
```
cd desktop
npm install
npm run dev
```
